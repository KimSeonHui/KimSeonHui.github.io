# Markdown 테스트

이 문서는 **remark**와 **remark-gfm**을 사용한 마크다운 변환 테스트를 위한 샘플 문서입니다.

## 텍스트 스타일링

*기울임체*와 **굵은 글씨**를 사용할 수 있습니다.
또한 ~~취소선~~도 지원됩니다.

## 목록

### 순서 없는 목록

- 항목 1
- 항목 2
  - 중첩 항목 2.1
  - 중첩 항목 2.2
- 항목 3

### 순서 있는 목록

1. 첫 번째 항목
2. 두 번째 항목
3. 세 번째 항목

### 체크 목록

- [x] 완료된 항목
- [ ] 미완료 항목
- [x] 또 다른 완료된 항목

## 인용문

> 인용문을 사용하면 다른 사람의 말이나 인용구를 표시할 수 있습니다.
>
> 여러 줄도 가능합니다.

## 코드

인라인 코드는 `const hello = 'world'`와 같이 표시됩니다.

코드 블록은 다음과 같이 표시됩니다:

```javascript
function greeting(name) {
  console.log(`Hello, ${name}!`);
}

greeting("World");
```
